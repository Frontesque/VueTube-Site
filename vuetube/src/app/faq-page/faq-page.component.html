FAQs

Frequently Asked Questions about VueTube

1. Will VueTube have the same features as Vanced? Yes, it will have all features from Vanced such as AdBlocking, RYD Integration, SponsorBlock and even more features like Privacy Protection. We are also planning on adding Twitch support to this app as well in the later future. 2. Why is VueTube app not working? This project is still in very early development but we will announce to everyone when it's ready to be used. 3. When will VueTube start working? As per what Front said a while ago, it will start to be usable with most basic YouTube features in the coming weeks. The entire app will probably be finished in about a month or two. 4. Where can I download the latest builds You can download the latest builds from either #apks (Android) and #ipas (iOS) in the Discord server.
The latest unstable builds are built via Github Actions 5. Is VueTube open source? Yes, the full source is available on the Github repository. 6. Will VueTube have the same UI as YouTube? Yes, it will have a similar UI to YouTube but also there will be lots of customization options to modify the UI to your likings. 7. Can VueTube be taken down by Google? Not sure if I can say the reason on why it might be taken down but we will add a feature to prevent Google from taking it down. 8. Will I be able to log in to get recommendations/subscriptions? The goal is to get login working, though whether this is through Micro-G or another way remains to be seen.
